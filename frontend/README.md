# 🎨 Frontend - Inventory Management System

This is the frontend of the inventory management system, built with **React, Vite, TailwindCSS, and Redux**.

## 🚀 Requirements
- **Node.js** v18+
- **npm** or **yarn**

## 🔧 Installation and Setup

### 1️⃣ Clone the Repository
```sh
git clone <REPO_URL>
cd frontend
```

### 2️⃣ Install Dependencies
```sh
npm install
```

### 3️⃣ Configure Environment Variables
Create a .env file in the frontend root and add:
# Server Configuration
PORT=3000


### 4️⃣ Start the Development Server
```sh
npm run dev
```

## 📄 Project Architecture
📂 src/
├── components/ → Reusable components
├── interfaces/ → Dto's
├── pages/ → Main pages
├── store/ → State management with Redux
├── services/ → API connection
├── App.tsx → Main application component

### 📌 Features
- Product Management (list, create, update, delete)
- Inventory Movements Registration
- Low Stock Alerts
- Modern Design with TailwindCSS v4.0
- State Management with useContext

## 🛠 Useful Commands
### 📌 Build the Application for Production

```sh
npm run build
```
